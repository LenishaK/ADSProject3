# ADSProject3 - Social Network Influence Analysis 

## Overview 
This project identifies key influencers in a social network using graph algorithms.

The system calculates an influence score based on the shortest path ditances between nodes.

## Features 
- Graph representation using adjaceny list
- BFS for unweighted networks
- Dijkstra's algorithm for weighted networks
- Influence score calculations
- CSV file input support

## How to run 
1. Open the project in visual studio
2. Ensure CSV files are in the correct directory
3. Run the program

## Example output 
Displays: 
- Influence scores for all nodes
- Highest and lowest influence nodes

## Technologies Used 
- C#
- .Net Console Application
